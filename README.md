# Data Structure Projects

A comprehensive collection of Java projects and assignments from the **Data Structures** course (Semester 4). This repository contains 6 main modules (Codelabs and Tasks) focusing on fundamental data structures and object-oriented principles.

## 🚀 Project Overview

### 1. Codelab 1: Generics Basics
Implementation of **Java Generics**. Demonstrates how to handle different data types in a single `GenericTicket` class.

### 2. Codelab 2: List & custom Linked List
- **Mission 1**: Demonstrates basic `ArrayList` operations (Jujutsu Kaisen theme).
- **Mission 2**: Custom implementation of a **Singly Linked List** (`GrandLineNavigation`) for route management (One Piece theme).

### 3. Codelab 3: Stack & Queue
Introduction to Java's built-in `Stack` and `Queue` collections for LIFO and FIFO management.

### 4. Tugas 1: Advanced Generics
A robust railway ticket system utilizing **multi-parameter generics** and **wildcards** to handle complex identity and booking types.

### 5. Tugas 2: Collection Management
- **Passenger Manager**: CRUD operations and linear search using `ArrayList`.
- **Schedule Manager**: Advanced usage of `LinkedList` and `Iterator` for train schedule manipulation.

### 6. Tugas 3: Practical Algorithms
- **Expression Converter**: Implements the **Shunting-Yard algorithm** for Infix to Postfix conversion using `Stack`.
- **Queue Service**: A visitor service system using a `Queue` for antrean and a `Stack` for transaction history.

## 🛠️ Tech Stack
- **Language**: Java
- **Core Concepts**: Generics, Collections Framework (ArrayList, LinkedList, Stack, Queue), Iterators, Object-Oriented Programming (OOP).
